# ComfyUI MultiAudio Mixer 🔊

A powerful multi-track audio mixing node for ComfyUI. Designed for cinematic sound design, lip-sync workflows, and complex audio-visual projects.

## Features
- **Multiple Audio Uploads:** Add as many tracks as you need.
- **Track Control:** Individual Volume, Pan (Balance), and Duration (Start/Stop) controls.
- **Precision Timing:** `Indent` field for exact start-time offsets (float seconds).
- **In-Node Player:** Preview each uploaded track directly in the ComfyUI interface.
- **Total Duration Calc:** Real-time calculation of the final audio length.

## Installation
1. Clone this repo to `ComfyUI/custom_nodes/`
2. Run `pip install -r requirements.txt`
3. Restart ComfyUI.
